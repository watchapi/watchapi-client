# Code of Conduct

## Our Pledge

We aim to provide a respectful, constructive, and professional environment for everyone who interacts with this project — including users, contributors, and maintainers.

Participation in this project implies agreement to follow this Code of Conduct.

---

## Expected Behavior

We expect all participants to:

- Be respectful and considerate in communication
- Assume good intent, even in disagreement
- Focus on constructive feedback and clear problem descriptions
- Respect differing viewpoints and experiences
- Accept maintainer decisions gracefully

---

## Unacceptable Behavior

The following behaviors are not acceptable:

- Harassment, insults, or personal attacks
- Discriminatory, abusive, or hostile language or behavior
- Trolling, spamming, or deliberately disruptive conduct
- Demanding support or features as an entitlement
- Repeatedly ignoring project guidelines or maintainer decisions

---

## Scope

This Code of Conduct applies to all project spaces, including but not limited to:

- GitHub issues and pull requests
- Discussions and comments
- Community forums or chats related to this project

---

## Enforcement

The project maintainer reserves the right to take appropriate action in response to unacceptable behavior, including:

- Warning the participant
- Closing or locking issues or pull requests
- Removing comments
- Temporarily or permanently restricting participation

These actions are taken to protect the health and focus of the project.

---

## Reporting Issues

If you experience or witness behavior that violates this Code of Conduct, you may report it privately by contacting the project maintainer.

All reports will be reviewed and handled at the maintainer’s discretion.

---

## Maintainer Authority

This project is maintained by a solo maintainer.
Decisions regarding scope, roadmap, and moderation are made by the maintainer and are final.

---

## Acknowledgment

This Code of Conduct is adapted for a small, focused open-source project and is intended to encourage respectful collaboration without unnecessary overhead.
